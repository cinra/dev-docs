# Vuepress for Project Template

Vuepress（静的サイトジェネレータ）による、プロジェクトテンプレート用のドキュメント置き場です。

## Requirements

```
node >= 8.x.x
npm >= 5.x.x
```

## Install

```shell
$ yarn install
```

## Start

```
$ yarn dev
# look at -> localhost:8080
```

## Deploy

```
$ git add .
$ git commit -am 'message'
$ git push
# published to -> docs.cinra.net
```
